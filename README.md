# Personal Config Files

## install

Step 1: Clone the configs

```shell
git clone git@github.com:chengruilin/config.git ~/.chrconfigs
```
or

```shell
git clone https://github.com/chengruilin/config.git ~/.chrconfigs
```

Step 2: Install the configs
``` shell
chmod +x install.sh
./install.sh
```

## zshrc

``` shell
plugins=(
  git
  // shell auto-complete
  zsh-autosuggestions
)

```
